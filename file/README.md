# 音频文件存储目录

这个目录用于存储语音合成服务生成的音频文件。

## 文件命名规则

- 格式：`audio_{reqId}_{timestamp}.{extension}`
- 示例：`audio_12345_20250905_143022.mp3`

## 文件类型

- **MP3**: 默认格式，兼容性最好
- **WAV**: 无损格式，文件较大
- **PCM**: 原始音频数据
- **OGG Opus**: 高效压缩格式

## 文件管理

- 文件会自动保存到此处
- 可以通过API下载、查询信息、删除文件
- 建议定期清理不需要的文件以节省磁盘空间

## API接口

- `GET /api/tts/audio/{reqId}` - 下载音频文件
- `GET /api/tts/audio/{reqId}/info` - 获取文件信息
- `DELETE /api/tts/audio/{reqId}` - 删除音频文件

## 注意事项

- 请确保有足够的磁盘空间
- 文件会永久保存，需要手动清理
- 建议在生产环境中配置定期清理策略
